## 📋 Sobre el Proyecto

Landing page moderna desarrollada con Next.js 16 para Peluquería Don Omar, barbería tradicional ubicada en Rosario, Rengo, Chile. El sitio presenta información de servicios, horarios, contacto y sistema de reservas vía WhatsApp.

**Características destacadas:**
- Diseño elegante con temática de barbería clásica
- Sistema de reservas integrado con WhatsApp
- Información detallada de servicios y precios
- Totalmente responsive (móvil, tablet, escritorio)
- Optimizado para SEO y rendimiento

## 🛠️ Stack Tecnológico

### Core
- **Next.js 16.1** - Framework de React con App Router
- **React 19** - Librería UI con Server Components
- **TypeScript 5** - Tipado estático y mejor DX

### Styling
- **Tailwind CSS 3.4** - Framework CSS utility-first
- **PostCSS** - Procesamiento de estilos
- **Animaciones CSS personalizadas** - Barber pole animado, efectos visuales

### Desarrollo
- **ESLint** - Linter de código
- **eslint-config-next** - Reglas específicas de Next.js

### Deployment
- **Vercel** - Hosting optimizado para Next.js con dominio personalizado

## 🚀 Desarrollo Local

### Prerrequisitos

```bash
Node.js 18+ 
npm, yarn, pnpm o bun
```

### Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/peluqueriadonomar-page.git
cd peluqueriadonomar-page

# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm run dev

# Abrir http://localhost:3000
```

### Scripts

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Inicia servidor de desarrollo en puerto 3000 |
| `npm run build` | Genera build de producción |
| `npm run start` | Ejecuta servidor de producción |
| `npm run lint` | Revisa código con ESLint |
